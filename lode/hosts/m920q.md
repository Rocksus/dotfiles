# m920q

Homelab server. Lenovo M920q, x86_64.

## Location

`hosts/m920q/`:
- `default.nix`, `configuration.nix`, `hardware-configuration.nix`, `disko-config.nix`, `secrets.nix`
- `services/`: `actualbudget.nix`, `audiobookshelf.nix`, `authentik.nix`, `containers/default.nix`, `homebox.nix`, `n8n.nix`, `paperless.nix`, `traefik.nix`

## Role

Self-hosted services hub. Reverse proxy via Traefik; identity via Authentik; containers via Podman.

## Wiring in flake

`nixosConfigurations.m920q` + `homeConfigurations."rocksus@m920q"`. Modules: disko, agenix, **authentik-nix** (only host with this).

## Services

| Service | Purpose |
|---|---|
| actualbudget | Actual Budget personal finance |
| audiobookshelf | Audiobook/podcast server |
| authentik | Identity provider (SSO) |
| homebox | Home inventory |
| n8n | Workflow automation |
| paperless | Document management |
| traefik | Reverse proxy / TLS termination |
| containers | Podman container definitions |
