# Lode Summary

A NixOS flake-based dotfiles repository managing four hosts (`t480`, `m920q`, `lv001`, `homelab-hz`) and the user `rocksus` via home-manager. The flake wires nixpkgs (unstable + stable 25.11), home-manager, disko, agenix, and several community inputs (ghostty, zen-browser, rust-overlay, swww, affinity-nix, authentik-nix, pre-commit-hooks). Host logic lives under `hosts/<name>` with shared config in `hosts/common`; user-facing features live under `home/features/{cli,desktop,editors,programming,browsers,design}` and are composed per-host via `home/rocksus/<host>.nix`. `m920q` is the homelab server running self-hosted services (actualbudget, audiobookshelf, authentik, homebox, n8n, paperless, traefik); `t480` is the primary laptop with Hyprland, wireguard, and DJ tools (mixxx). Secrets are managed with agenix; disk provisioning uses disko via `make disko-install`.
